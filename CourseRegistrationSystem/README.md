# Course-Registration-System
A Java-based Course Registration Management System developed in NetBeans. This is an OS project implementing FCFS, SJF, and Round Robin scheduling algorithms. It provides a GUI for course registration and enrollment management. Awarded 2nd place at university project exhibition.


# Overview
A Java-based Course Registration Management System developed in NetBeans.  
This is an Operating Systems (OS) project that implements CPU scheduling algorithms such as FCFS (First Come First Serve), SJF (Shortest Job First), and Round Robin.

The system provides a simple GUI for students to register for courses and manage enrollments efficiently.

OS Concepts Used
- First Come First Serve (FCFS) Scheduling
- Shortest Job First (SJF) Scheduling
- Round Robin Scheduling


# Features
- SignIn
- Register Course
- Enroll in Additional Courses
- Update Courses
- Other features such as View Report Card, View Attendance, View Quiz+Assignment marks. 


# Technologies Used
- Java
- NetBeans IDE
- Swing GUI

# Achievement
This project was awarded 2nd Place at a university Software Engineering project exhibition.

1. Open the project in NetBeans
2. Build the project
